# RankEval Studio

AI output quality evaluation and ranking tool built with React, Vite, Firebase Firestore, and Vercel.

## Features
- Experiment builder for AI response evaluation
- Anonymous rater interface
- Ranking metrics dashboard: NDCG@K, MRR, Precision@K, Cohen's Kappa
- CSV export

## Run locally
1. npm install
2. Add Firebase config to `.env`
3. npm run dev

## Deploy
Connect the repo to Vercel and set env vars in the project dashboard.
